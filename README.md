# case-studies
A collection of frontend case studies featuring React, Tailwind CSS, API integration, responsive design, performance improvements, and production-ready UI.

### Arcanova - HubSpot CMS Website <br>
##### Live URL: https://www.arcanova.no/<br>
About Arconova:  Arcanova is a modern real estate website designed for property selling and rentals. 
Built on HubSpot CMS using HubL and HubDB, this project was developed with HTML, CSS, JavaScript, and custom libraries. It supports dynamic property listing pages, structured data management, and scalable content updates—allowing the client to manage listings without ongoing developer support.
